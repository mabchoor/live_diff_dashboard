# Live Diff Dashboard

## Overview
A real-time visualization tool for comparing and tracking differences in data, code, or documents. This project provides an interactive dashboard for monitoring changes live.

## Features
- **Live Monitoring**: Real-time tracking of differences
- **Interactive Dashboard**: User-friendly interface for visualizing changes
- **Python-based**: Built with Python and Jupyter notebooks
- **Flexible**: Easily adaptable for different types of comparisons

## Installation

### Requirements
- Python 3.7+
- Jupyter Notebook

### Setup
```bash
# Clone the repository
git clone https://github.com/mabchoor/live_diff_dashboard.git
cd live_diff_dashboard

# Install dependencies
pip install -r requirements.txt
```

## Usage

1. Open Jupyter Notebook:
```bash
jupyter notebook
```

2. Open `Untitled0.ipynb` to start using the dashboard

3. Run the cells to initialize and visualize the differences

## Project Structure
```
├── live_diff_dashboard.py    # Main Python module
```
